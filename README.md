# CSSGB Study Notes

Source-grounded study notes for the **ASQ Certified Six Sigma Green Belt (CSSGB)** exam, covering all 65 Body of Knowledge topics across the full DMAIC cycle.

**[View the site](https://duymlcoding.github.io/cssgb-notes/)**

---

## Contents

| Phase | Topics |
|-------|--------|
| I — Overview: Six Sigma and the Organization | 8 |
| II — Define | 23 |
| III — Measure | 13 |
| IV — Analyze | 6 |
| V — Improve | 6 |
| VI — Control | 10 |

Each note follows the official BoK structure, is grounded in the ASQ CSSGB Handbook (2nd ed.) and CSSC Training Manual (2018), and closes with targeted exam tips keyed to its Bloom's Taxonomy level.

Statistical topics (distributions, capability indices, hypothesis tests, regression, MSA) include display-math formulas and fully worked numerical examples.

## Local preview

```bash
npm install -g mystmd
myst start
```

Then open `http://localhost:3000`.

## Build

```bash
myst build --html
```

Output goes to `_build/html/`. GitHub Actions deploys automatically on every push to `main`.
